Installation

Step 1. Download jQuery Lightbox, and extract it to your hard drive

Step 2. Move the extracted jquery-lightbox directory to your website's root/home folder

Step 3. Include jQuery (insert into your page's head tag)

<!-- Include jQuery (Lightbox Requirement) --> <script type="text/javascript" src="http://www.yoursite.com/jquery-lightbox/scripts/jquery-1.4.2.min.js"></script>

Step 4. Include jQuery Lightbox (insert into your page's head tag)

<!-- Include Lightbox (Production) --> <script type="text/javascript" src="http://www.yoursite.com/jquery-lightbox/scripts/jquery.lightbox.min.js"></script>

Usage

Introduction

There are a few ways you can use jquery lightbox. Firstly, let's specify our gallery we will be using.

<div class="gallery"> <ul class="images"> <li class="image"> <a href="http://farm2.static.flickr.com/1395/1116205566_42ce0841ab.jpg" title="View Image"> <img title="tallest, deepest" src="http://farm2.static.flickr.com/1395/1116205566_42ce0841ab_s.jpg" height="75" width="75"> </a> </li> <li class="image"> <a href="http://farm1.static.flickr.com/64/230712309_392c8d5bb7_o.jpg" title="View Image"> <img title="Rolling Clouds" src="http://farm1.static.flickr.com/64/230712309_392c8d5bb7_s.jpg" height="75" width="75"> </a> </li> </ul> </div>

Making it work with lightbox

All we need to do, is add the CSS class lightbox to our gallery. We do this by changing:

<div class="gallery">

To:

<div class="gallery lightbox">

And that's all you need to do, when you reload the page your gallery will now use the lightbox script.

Using existing class names

But what if you would like to keep the class names intact, and not have to add the lightbox class?

You can do this by running the following instead:

<script type="text/javascript"> $('.gallery').lightbox(); </script>

And that will do it for you! You can also change the text .gallery which we pass to the scan function to whatever selector you would like to use for your lightboxes.

Custom Usage

Attaching links to your lightbox

As we can see in the example at the top of the page, we allow for links in our lightboxes. To do this, we now improve our gallery to a structure which includes enough semantic information so we know what to do:

<div class="gallery"> <ul class="images"> <li class="image"> <a href="http://farm2.static.flickr.com/1395/1116205566_42ce0841ab.jpg" class="image-large image-link" title="View Large"></a> <a href="http://www.flickr.com/photos/maybemaq/1116205566" class="image-thumbnail image-link" title="View Image"> <img title="tallest, deepest" src="http://farm2.static.flickr.com/1395/1116205566_42ce0841ab_s.jpg" height="75" width="75"> </a> <div class="image-title"> tallest, deepest </div> <div class="image-description"> south perth daydream; or you prefer <a href="http://www.flickr.com/photos/maybemaq/1092924860/" target="_blank">night version</a>? </div> </li> <li class="image"> <a href="http://farm1.static.flickr.com/64/230712309_392c8d5bb7_o.jpg" class="image-large image-link" title="View Large"></a> <a href="http://www.flickr.com/photos/richardgiles/230712309" class="image-thumbnail image-link" title="View Image"> <img title="Rolling Clouds" src="http://farm1.static.flickr.com/64/230712309_392c8d5bb7_s.jpg" height="75" width="75"> </a> <div class="image-title"> Rolling Clouds </div> <div class="image-description"></div> </li> </ul> </div>

Notice how we now have multiple a elements in our li.image element. These are the links for our image.

All that is required is to add the CSS class image-link to any a element which we would like to become a link in our lightbox.

We also add a title attribute to the a elements, these will be the link text in our lightbox.

Including Descriptions

As we can see in the example at the top of the page, we allow for descriptions in our lightboxes. To do this, we now improve our gallery to a structure which includes enough semantic information so we know what to do:

<div class="gallery"> <ul class="images"> <li class="image"> <a href="http://farm2.static.flickr.com/1395/1116205566_42ce0841ab.jpg" class="image-large image-link" title="View Large"></a> <a href="http://www.flickr.com/photos/maybemaq/1116205566" class="image-thumbnail image-link" title="View Image"> <img title="tallest, deepest" src="http://farm2.static.flickr.com/1395/1116205566_42ce0841ab_s.jpg" height="75" width="75"> </a> <div class="image-title"> tallest, deepest </div> <div class="image-description"> south perth daydream; or you prefer <a href="http://www.flickr.com/photos/maybemaq/1092924860/" target="_blank">night version</a>? </div> </li> <li class="image"> <a href="http://farm1.static.flickr.com/64/230712309_392c8d5bb7_o.jpg" class="image-large image-link" title="View Large"></a> <a href="http://www.flickr.com/photos/richardgiles/230712309" class="image-thumbnail image-link" title="View Image"> <img title="Rolling Clouds" src="http://farm1.static.flickr.com/64/230712309_392c8d5bb7_s.jpg" height="75" width="75"> </a> <div class="image-title"> Rolling Clouds </div> <div class="image-description"></div> </li> </ul> </div>

Notice how we now have some div elements in our li.image element. These are the title and description for our image.

All that is required is to add the these elements anywhere within our li.image (we chose to have the image's title in the a element, but this doesn't have to be the case), and insert our title and description into them.

Styling your gallery

While not part of the lightbox script at all, here is some nice CSS so you can style your gallery like our example above.

/* Gallery Styling */ .gallery .image-title, .gallery .image-description { display:none; } .gallery { width: 100%; text-align: center; margin-left: auto; margin-right: auto; padding: none; } .gallery ul.images { list-style-type: none; border: none; padding: none; } .gallery ul.images li.image { display: inline; line-height: 0; } .gallery ul.images li.image a.image-thumbnail { text-decoration: none; color: inherit; cursor: pointer; margin: 14px; display: inline-block; background: white; padding: 3px; line-height: 0; -webkit-border-radius: 4px 4px; -moz-border-radius: 4px 4px; border-radius: 4px 4px; -moz-box-shadow: 0px 0px 8px #AAA; -webkit-box-shadow: 0px 0px 8px #AAA; box-shadow: 0px 0px 8px #AAA; -webkit-transition: -webkit-box-shadow 0.1s ease-out; -moz-transition: -webkit-box-shadow 0.1s ease-out; -o-transition: -webkit-box-shadow 0.1s ease-out; transition: -webkit-box-shadow 0.1s ease-out; } .gallery ul.images li.image a.image-thumbnail img { border: none; padding: none; -webkit-border-radius: 4px 4px; -moz-border-radius: 4px 4px; border-radius: 4px 4px; } .gallery ul.images li.image a.image-thumbnail:hover { -moz-box-shadow: 0px 0px 8px #222; -webkit-box-shadow: 0px 0px 8px #222; box-shadow: 0px 0px 8px #222; }
